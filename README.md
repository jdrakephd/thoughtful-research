# thoughtful-research
Repo for analysis related to Thoughtful Research (Erin Sowell, Marcus Cunha, et al)

## Repository Structure

This repository is organized to hold both white papers and specific sub-project analyses. Each unique item has its own subdirectory for better organization and maintainability.

### White Papers (`white-papers/`)

Contains comprehensive research reports and white papers. Each white paper has its own subdirectory with all related materials including source documents, data, analysis scripts, and visualizations.

See [white-papers/README.md](white-papers/README.md) for detailed guidelines.

### Sub-Project Analyses (`analyses/`)

Contains specific project analyses and research studies. Each analysis has its own subdirectory with data, scripts, results, and documentation.

See [analyses/README.md](analyses/README.md) for detailed guidelines.

## Getting Started

To add a new white paper or analysis:

1. Create a new subdirectory in the appropriate folder (`white-papers/` or `analyses/`)
2. Use a descriptive name with lowercase letters and hyphens (e.g., `market-analysis-2026`)
3. Add a `README.md` in your subdirectory explaining the project
4. Organize your work following the structure guidelines in the respective README files

## Contributing

When adding new work:
- Follow the naming conventions and structure guidelines
- Include a comprehensive README.md in each project subdirectory
- Document your data sources and methodology
- Keep related files organized within the project subdirectory
