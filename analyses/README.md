# Sub-Project Analyses

This directory contains specific sub-project analyses produced by Thoughtful Research.

## Organization

Each sub-project analysis should have its own subdirectory with a descriptive name. For example:

```
analyses/
├── customer-segmentation-study/
├── competitor-analysis-q1-2026/
└── product-market-fit-evaluation/
```

## Naming Convention

- Use lowercase letters
- Separate words with hyphens (-)
- Use descriptive names that clearly indicate the project
- Include dates or quarters if relevant

## Structure

Each analysis subdirectory should typically contain:

- `README.md` - Overview, objectives, and key findings
- Data files
- Analysis scripts or notebooks
- Results and reports
- Visualizations
- Documentation

## Example

```
analyses/customer-segmentation-study/
├── README.md
├── data/
│   ├── raw/
│   └── processed/
├── scripts/
│   ├── data_cleaning.py
│   └── segmentation_analysis.py
├── notebooks/
│   └── exploratory_analysis.ipynb
├── results/
│   ├── report.pdf
│   └── findings.md
└── figures/
    ├── segment_distribution.png
    └── key_metrics.png
```
